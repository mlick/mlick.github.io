# mlick.github.io
**利用的是[Hexo](https://github.com/hexojs/hexo)这个框架，主题使用的是[nexT](http://theme-next.iissnan.com/)**
## 安装
从github上克隆到本地
```
git clone https://github.com/mlick/mlick.github.io.git
```
切换至source分支 
```
git checkout source
```

安装所需要的库 
```
nmp install
```
安装所需要的库
    nmp install
## 运行
    hexo server
## 结果
    http://localhost:4000/
